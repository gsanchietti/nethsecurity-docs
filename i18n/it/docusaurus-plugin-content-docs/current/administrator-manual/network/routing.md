# Rotte statiche {#static_routes-section}

Il routing statico in Linux consente un controllo preciso su come i pacchetti di dati navigano attraverso una rete. A differenza del routing dinamico, che è automatico, il routing statico richiede una configurazione manuale da parte degli amministratori di rete.

Il routing statico è semplice e prevedibile, rendendolo ideale per reti stabili in cui i percorsi non cambiano frequentemente. Gli amministratori possono definire le rotte manualmente, fornendo istruzioni specifiche su come i dati devono viaggiare attraverso la rete.

Le rotte statiche possono essere configurate dalla pagina `Routes`, nella sezione `Network` dal menu a sinistra.

Per aggiungere una nuova rotta fare clic sul pulsante **Add route**. Per modificare una rotta esistente fare clic sul pulsante **Edit** nel record della tabella. Per eliminare una rotta, fare clic sul menu con tre punti **⋮** nel record della tabella e premere il pulsante **Delete**.
